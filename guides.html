Creating a neural network

The NeuralNetwork class is LANN's representation of a neural network. To create a NeuralNetwork, you must use the NeuralNetwork.Builder class. Below is an example of how to create a simple neural network with 3 input neurons, 1 hidden layer of 2 neurons, and an output layer of 1 neuron.

NeuralNetwork net = new NeuralNetwork.Builder()
    .addLayer(2, 3, new Linear())
    .addLayer(3, 1, new Sigmoid())
    .build();

You can have as many layers as needed. The function header is:

NeuralNetwork.Builder.addLayer(int input, int output, Activation activationFunction)

There are several different pre-defined activation functions, all of which are listed below.

Binary
LeakyReLU
Linear
ReLU
Sigmoid
Softmax
Softplus
Tanh

If you are looking to do classification, then the Softmax activation function is the class you are looking to use for the output layer. This will give you a probability distribution of your classes.

After creating the neural network, the weights for each neuron will be random, so you will need to train the network to get a useful prediction.

You can get the output of the network by using the method shown below.

Matrix input = new Matrix(new double[][]{{0.5}, {0.3}});
Matrix output = net.predict(input);

Notice the use of the Matrix class for representing input and output. You can find out more about this class by reading the Matrix library section of this guide.

Matrix library

LANN comes bundled with a Matrix library for Java. This library contains a class, Matrix, which represents a mathematical matrix.

To create a Matrix object, you can use one of the following methods.

Matrix matA = new Matrix(10, 8); // Creates a 10 by 8 matrix (10 rows, 8 columns) filled with 0s.
Matrix matB = new Matrix (10, 8, 0.1); // Creates a 10 by 8 matrix (10 rows, 8 columns) filled with 0.1 in each position.
Matrix matC = new Matrix(new double[][]{{1.2, 4}, {5, 9}, {0.1, -0.8}}); // Creates a 3 by 2 matrix (3 rows, 2 columns) of user defined values.

You can also do some fancy calculations with the matrices.

Matrix matATrans = matA.transpose();
Matrix matAMultiply = matA.multiply(matB);
Matrix matBSubtract = matB.subtract(matA);
Matrix matBScalarSubtr = matB.subtract(5);
Matrix matCScalarMult = matC.multiply(6);
Matrix matBDot = matB.dot(matATrans);
Matrix oneHotC = matC.oneHot();
Matrix normC = matC.normalize();
double max = matC.max();
double min = matA.min();
int[] rowColOf5 = matC.find(5);
Matrix applyFunctionA = matA.map(new Matrix.Function(){
    @Override
    public double function(double x) {
        return Math.abs(x);
    }
});

LANN includes this matrix library because all of the input and outputs into the NeuraNetwork object will be Matrix objects.

To use this library seperately, you can download it from GitHub.
